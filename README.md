# Desenvolvimento Web Front-End: HTML5 e CSS3

>  **Lobby de Projetos:** Acesse o ambiente online onde todos os exercícios deste repositório estão: **[Visualizar Projetos (GitHub Pages)](https://liviapinheiro.github.io/html-css/)**

Este repositório contém meus códigos, projetos e exercícios práticos desenvolvidos durante as aulas de HTML5 e CSS3 do , servindo como um complemento prático à minha formação no curso de Análise e Desenvolvimento de Sistemas.

O objetivo principal deste espaço é documentar minha evolução na construção de interfaces web, garantindo o aprendizado de layouts semânticos, acessíveis e responsivos.

## Tópicos Abordados

Durante esta primeira parte do curso foram abordados os seguintes tópicos:

* **Fundamentos de HTML5:** Estruturação semântica de páginas, uso correto de tags, formatação de textos e inserção de mídias.
* **Estilização com CSS3:** Manipulação de cores, tipografia, bordas, sombras e margens.
* **Layouts Modernos:** Boas práticas para construir uma interface responsiva, acessível e bem estruturada.

## Tecnologias Utilizadas

* **Linguagens:** HTML5 e CSS3
